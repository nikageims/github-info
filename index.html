GitHub Desktop მიმოხილვა

როგორ ვმართოთ პროექტი მარტივად, კოდის გარეშე

01

პროექტის კლონირება (Clone)

ეს არის პირველი ნაბიჯი: ონლაინ რეპოზიტორიის გადმოწერა შენს კომპიუტერში.

  • გახსენი GitHub-ში სასურველი რეპოზიტორია.
  • დააჭირე მწვანე ღილაკს Code.
  • აირჩიე Open with GitHub Desktop.
  • მიუთითე მისამართი შენს კომპიუტერში და დააჭირე Clone.
02

ცვლილებების შენახვა (Commit)

კოდში ცვლილების შეტანის შემდეგ, ის უნდა აღწერო და შეინახო ისტორიაში.

  • მარცხენა პანელში მონიშნე შეცვლილი ფაილები.
  • Summary ველში დაწერე მოკლე სათაური.
  • დააჭირე ლურჯ ღილაკს Commit to main.
03

სად ვნახოთ უკვე გაკეთებული Commits

შენს მიერ შენახული ისტორიის ნახვა ნებისმიერ დროს შეგიძლია როგორც პროგრამაში, ისე საიტზე.

  • GitHub Desktop-ში აირჩიე ტაბი History.
  • ონლაინ სანახავად გადადი შენს GitHub რეპოზიტორიაში.
  • მოძებნე ღილაკი Commits საათის ხატულასთან ერთად.
