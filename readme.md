# 🇬🇧 English Version

🌐 **ქართული ვერსია ზემოთ ⬆️**

This is an interactive web presentation designed to help users easily learn how to manage projects using **GitHub Desktop**, without writing complex code.

## 🚀 Live Demo
You can view the website here:  
👉 https://profound-pegasus-586f42.netlify.app/

## ✨ Project Features

- **Step-by-step instructions** – The process is divided into clear steps (01, 02, 03)  
- **Visual explanation** – Each step includes screenshots for better understanding  
- **Simple language** – Designed for beginners who want to start working with Git  

## 🛠️ What will you learn?

1. **Clone** – How to download an online repository to your computer  
2. **Commit** – How to describe and save changes in your project history  
3. **History** – Where and how to view previous `commits` on GitHub  

## 💻 Technologies Used

- **HTML5** – Website structure  
- **CSS3** – Styling and layout  
- **GitHub Desktop** – Main learning tool
