# GitHub Desktop - სრული ვიზუალური სახელმძღვანელო

ეს არის ინტერაქტიული ვებ-პრეზენტაცია, რომელიც შექმნილია იმისათვის, რომ მომხმარებლებს მარტივად, კოდის წერის გარეშე ასწავლოს პროექტების მართვა **GitHub Desktop**-ის გამოყენებით.

## 🚀 ცოცხალი დემო
შეგიძლიათ იხილოთ საიტი აქ: [GitHub Desktop Tutorial](https://profound-pegasus-586f42.netlify.app/)

## ✨ პროექტის მახასიათებლები
- **ეტაპობრივი ინსტრუქცია:** პროცესი დაყოფილია ნაბიჯებად (01, 02, 03).
- **ვიზუალური მხარე:** თითოეულ ნაბიჯს ახლავს შესაბამისი სკრინშოტი უკეთესი აღქმისთვის.
- **მარტივი ენა:** განკუთვნილია დამწყებთათვის, ვისაც სურს ვერსიების კონტროლის სისტემასთან (Git) მუშაობის დაწყება.

## 🛠️ რას ისწავლით ამ სახელმძღვანელოთი?
1. **Clone (კლონირება):** როგორ გადმოვწეროთ ონლაინ რეპოზიტორია საკუთარ კომპიუტერში.
2. **Commit (შენახვა):** როგორ აღვწეროთ კოდში შეტანილი ცვლილებები და შევინახოთ ისტორიაში.
3. **History (ისტორია):** სად და როგორ ვნახოთ უკვე შესრულებული `commits` GitHub-ზე.

* **GitHub Desktop** - როგორც ძირითადი სასწავლო ინსტრუმენტი.
